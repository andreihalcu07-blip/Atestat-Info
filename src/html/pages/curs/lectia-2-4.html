Modul 2 — Electronică de Bază

Lecția 2.4 — MOSFET și VRM

Cum MOSFET-urile construiesc regulatoarele de tensiune care dau viață consolei.

Începe Lecția

Introducere

VRM (Voltage Regulator Module): Circuit cu MOSFET-uri care convertește tensiune de intrare (12V) în tensiuni de ieșire precise (1.05V, 3.3V, etc.) cu stabilitate extremă.

PS5 are ~15-20 module VRM pe placa de bază. Fiecare în-și joacă rolul: CPU core, GPU core, RAM, I/O.

VRM-ul e "creierul" care menține tensiuni stabile. Dacă VRM crapă, consola nu pornește.

Teorie Structurată

2.1 — Principiu VRM: Buck Converter

Buck Converter: Schimă rapid între ON și OFF a MOSFET-ului pentru a crea tensiune medie mai mică decât intrare.

V_out = V_in × (Ton / (Ton + Toff))

  • Ton: Timp tranzistor ON
  • Toff: Timp tranzistor OFF
  • Ration: Raport de comutare (duty cycle)

Exemplu: 12V input, comutare 50/50 → output 6V medie. 30/70 → output 3.6V.

2.2 — Componente VRM

  • MOSFET High/Low (H-L): 2N+1 MOSFET-uri comutând alternativ. N-channel și P-channel în punte.
  • Inductanță (bobina): Stochează energie, furnizează curent continuu la ieșire. 1µH-10µH tipic.
  • Condensator ieșire: ESR mic pentru a filtra ripple. 100µF+.
  • Controller IC: PWM generator care orchestrează comutarea MOSFET-urilor. ex: IR3570, TI TPS54200+.
  • Feedback & sensori: Măsoară tensie și curent, ajustează duty cycle pentru a menține V_out stabil.

2.3 — Controlul Feedback

VRM măsoară tensiunea de ieșire (1.05V dorită) și o compară cu o referință internă. Dacă V_out < V_ref, crește duty cycle (T_on mai lung). Dacă V_out > V_ref, reduce duty cycle.

Feedback divider: Doi rezistori (R1, R2) împart V_out. R1//R2 contribuie la V_feedback care se compară cu V_ref.

Ripple Control: VRM modern detectează ripple (milivolți de fluctuație rapida) și ajustează comutare pentru a-l reduce.

2.4 — Performanță VRM: Eficiență și Ripple

Eficiență: Putere de ieșire / Putere de intrare. Modern VRM: 85-92% (din cauza pierderi MOSFET, inductanță, condensator ESR).

Ripple: Oscilație în tensiune de ieșire. Calitate VRM: ripple < 20mV. Slabă: > 50mV.

Respons time: Cât de repede VRM reacționează la schimbări de curent. Sub-microsecunde pentru VRM modern.

2.5 — Probleme VRM și Degradare

Condensator electrolitic defect: Capacitate pică, ESR crește, ripple crește. Instabilitate CPU.

MOSFET ars: On-resistance crește (posibil 10x din cauza deteriorării oxide). Putere disipata nu poate fi evacuată, temperatura crește necontrolat. Avalanșă termică.

Bobina cu miez crăpat: Inductanță cade, ripple crește.

Controller IC defect: PWM nu mai orchestrează; VRM furnizează tensiune aleatorie.

Legătura Fizică – Informatică

Fizica: VRM folosește inducție (bobina stochează) și comutare rapidă (MOSFET) pentru a converti și regula tensiuni.

Informatica: CPU necesită 1.05V exact. Firmware ajustează voltaj pentru stabilitate. VRM livrează. Tensiune instabilă → logic se rupe.

Aplicare Directă în Console

Exemplu: CPU Rails PS5

2 VRM dedicate pentru CPU core: 1.05V. 5 faze per VRM × 2 = 10 faze total. Fiecare fază cu 10 MOSFET-uri. 100 MOSFET-uri total pe CPU rail!

Curent: 90A total. Distribuția: 90A / 10 faze = 9A per fază. Fiecare MOSFET 9A / 10 = 0.9A. Curent mic per MOSFET → pierderi mici, siguranță mare.

Exemplu Real: Design VRM PS5

Calculare duty cycle: V_out = 1.05V, V_in = 12V.

Duty cycle = V_out / V_in = 1.05 / 12 = 8.75%.

Deci MOSFET e ON 8.75% din timp.

Calcul Ripple: Frekvență comutare 130kHz. Inductanță 5µH.

Ripple = (V_in × Duty) / (L × f)
Ripple = (12V × 0.0875) / (5µH × 130kHz) ≈ 16.15mV

Calitate bună.

Probleme Frecvente

⚠️ Consola nu pornește (No power)

Potența: VRM furnizează 0V din cauza MOSFET defect sau controller IC mort.

⚠️ Consola reboot-uri aleatorii

Cauza: Ripple VRM prea mare din cauza condensator defect. CPU instabil.

⚠️ VRM se încălzește anormal

Cauza: MOSFET degradat (R_DS crescut), pierderi mari.

Recapitulare

  • VRM (Voltage Regulator):Convertor de tensiune cu MOSFET comutând.
  • Buck converter: Scade tensiune prin comutare rapidă.
  • Duty cycle: Raport ON/OFF care determină V_out.
  • Componente: MOSFET, bobina, condensator, controller IC, feedback.
  • Eficiență: 85-92% modern.
  • Ripple: < 20mV bun, > 50mV slab.
  • PS5: 15-20 module VRM. CPU: 100+ MOSFET-uri.

Exercițiu Aplicat

🧠 Exercițiu: Proiect Simplificat VRM pentru CPU

Date: V_in = 12V, V_out dorită = 1.05V, I_out = 10A, frecvență = 130kHz, L = 2µH.

  1. Calculează Duty Cycle
  2. Calculează Ripple de curent (ΔI = V_out / L × Δt)
  3. Dacă condensator C_out = 1000µF, calculează ripple de tensiune (ΔV = ΔI / C / f)
  4. E ridicat ripple-ul? (< 20mV = bine)
  5. Cum ajustezi pentru a reduce ripple? (Sugestii)
Arată rezolvarea

1. Duty = V_out / V_in = 1.05 / 12 = 8.75%

2. Δt = Duty / f = 0.0875 / 130kHz ≈ 673ns. ΔI = 1.05V / 2µH × 673ns = 352.6mA.

3. ΔV = ΔI / (C × f) = 0.3526A / (1000µF × 130kHz) ≈ 2.7mV.

4. 2.7mV e extrem de bine! < 20mV. Excelență.

5. Pentru a reduce mai mult: creștere L, creștere C, creștere f, sau feedback control mai agresiv.
